# site-styx

外部站点的头尾导航，及全局样式

## 文件夹

npm start

http://localhost:3210

```site-styx
    /-
        src/    // 前端源码
        config/    // webpack 配置
        demo/    // demo server 源码 (指定端口号 || 3210)
        .gitignore    // git 忽略
        nodemon.json    // nodemon 配置
        package.json    // 依赖配置
        README.md    // read me
        tsconfig.front.json    // front's tsconfig
        tsconfig.json    // server's tsconfig
```
